Introduction

This project is developed for:
(1) Building and visualizing business process which is represented in Business Process Modeling Notation (BPMN) UML diagrams, Softgoal Interdependency Graph (SIG), and Problem Interdependency Graph (PIG).
(2) Improving business process through calculating Key Performance Index (KPI) from stored data in Cassandra NoSQL big data database and data analysis by Spark machine learning APIs for prediction.
